# Сценарий демонстрации на защите (5-7 минут)

## 0. Подготовка (до начала выступления)

1. Активировать окружение: `source .venv/bin/activate`.
2. Запустить: `python web_app.py`.
3. Открыть `http://127.0.0.1:8000`.
4. Оставить в редакторе пример `simple_box`.

## 1. Краткое вступление (30-40 секунд)

- Цель: показать модуль параметрического 3D-моделирования с инженерными расчетами.
- Подчеркнуть: модель задается сценарием и автоматически пересчитывается при изменении параметров.

## 2. Базовый кейс `simple_box` (1 минута)

1. Нажать `Построить модель`.
2. Показать 3D-окно, вращение и масштаб.
3. Показать объем, массу и центр масс.
4. Нажать `Скачать STL`.

## 3. Параметрический пересчет (1.5 минуты)

1. В `simple_box` изменить `param a` и `param b`.
2. Нажать `Построить модель`.
3. Показать, что геометрия и инженерные характеристики обновились.
4. Показать таблицу параметров.

## 4. Более сложный пример `bracket` (1.5 минуты)

1. Выбрать шаблон `bracket.txt`.
2. Нажать `Построить модель`.
3. Показать составную геометрию.
4. Нажать `Экспорт отчёта CSV`.

## 5. Работа с проектом (1 минута)

1. Нажать `Сохранить проект`.
2. Очистить часть текста в редакторе.
3. Нажать `Загрузить проект` и восстановить сценарий.

## 6. Завершение (30 секунд)

- Подчеркнуть, что реализованы: параметризация, сценарный интерфейс, пересборка, инженерные расчеты, экспорт.
- Перечислить направления развития: булевы операции, STEP, расширение языка выражений.
